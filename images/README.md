## How to Download Images

### Option 1: Manual Download
You can manually download each image by visiting the URLs above and saving them with the corresponding filename.

### Option 2: Using PowerShell Script
Run the `download-images.ps1` script (see below) to automatically download all available images.

### Option 3: Use Placeholder Images
For quick testing, you can use placeholder images from services like:
- https://via.placeholder.com/[width]x[height]
- https://placehold.co/[width]x[height]

---

## Image Specifications

- **Logo**: PNG format, transparent background recommended
- **Hero/Banners**: JPG format, high quality (1920px wide minimum)
- **Icons**: PNG format with transparency, 150x150px or larger
- **Gallery**: JPG format, 600x400px minimum
- **Team Photos**: JPG format, square format (500x500px minimum)

---

## Notes
- All images should be optimized for web use (compressed without significant quality loss)
- Maintain aspect ratios when resizing
- Use WebP format for better compression if browser compatibility allows
- Ensure all images have appropriate alt text for accessibility (already implemented in HTML)
